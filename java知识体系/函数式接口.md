## 1. **Consumer<T> - 消费者**
**用途**：接收参数执行操作，无返回值
java
// 遍历集合
List<String> list = Arrays.asList("A", "B", "C");
list.forEach(s -> System.out.println(s));  // Consumer

// 批量处理
Consumer<String> processor = s -> {
    String result = s.toLowerCase();
    saveToDatabase(result);
};
list.forEach(processor);

#  **Function<T,R> - 函数**

**用途**：数据转换/映射，有输入和输出

java

// 字符串转换
Function<String, Integer> strToInt = s -> Integer.parseInt(s);

// 数据转换链
Function<String, String> trim = String::trim;
Function<String, String> toUpper = String::toUpperCase;
Function<String, String> pipeline = trim.andThen(toUpper);

// Stream中的map操作
List<String> names = Arrays.asList(" alice ", " bob ");
List<String> cleaned = names.stream()
    .map(trim.andThen(toUpper))
    .collect(Collectors.toList());

## 3. **Predicate<T> - 断言**

**用途**：条件判断，返回布尔值

java

// 过滤数据
Predicate<Integer> isEven = n -> n % 2 == 0;
Predicate<String> isLong = s -> s.length() > 5;

// 组合条件
Predicate<String> isValid = isLong.and(s -> s.contains("@"));
List<String> emails = emailsList.stream()
    .filter(isValid)
    .collect(Collectors.toList());

## 4. **Supplier<T> - 供应商**

**用途**：生成/提供数据，无参数有返回值

java

// 延迟初始化
Supplier<ExpensiveObject> supplier = () -> {
    System.out.println("创建对象");
    return new ExpensiveObject();
};

// 工厂模式
Supplier<List<String>> listFactory = ArrayList::new;
Supplier<LocalDateTime> timeSupplier = LocalDateTime::now;

// 懒加载
class LazyCache {
    private Supplier<Data> dataSupplier = () -> loadFromDB();
    private Data cached;
    
    public Data get() {
        if (cached == null) {
            cached = dataSupplier.get();
        }
        return cached;
    }
}

## 5. **Runnable - 可运行任务**

**用途**：定义线程执行的任务

java

// 创建线程任务
Runnable task = () -> {
    System.out.println("线程执行中...");
    // 执行耗时操作
};

// 线程池执行
ExecutorService executor = Executors.newFixedThreadPool(3);
executor.submit(task);
executor.submit(() -> System.out.println("另一个任务"));

// 定时任务
ScheduledExecutorService scheduler = Executors.newScheduledThreadPool(1);
scheduler.scheduleAtFixedRate(
    () -> System.out.println("定时执行"),
    0, 1, TimeUnit.SECONDS
);

## 6. **Comparator<T> - 比较器**

**用途**：定义排序规则

java

// 自定义排序
List<Person> people = getPeople();

// 传统方式
Comparator<Person> byAge = (p1, p2) -> Integer.compare(p1.getAge(), p2.getAge());

// 方法引用简化
Comparator<Person> byName = Comparator.comparing(Person::getName);
Comparator<Person> byAgeThenName = Comparator
    .comparing(Person::getAge)
    .thenComparing(Person::getName);

// 使用
people.sort(byAgeThenName);

// 复杂排序：null安全的降序
Comparator<Person> safeComparator = Comparator
    .nullsLast(Comparator.comparing(Person::getSalary))
    .reversed();
