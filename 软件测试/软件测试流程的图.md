# 活动图
## 组成
* 动作 （圆角矩形（如 “输入账号”））
* 泳道 （分不同角色的操作区域（如 “用户”“系统”））
* 分支/合并 （菱形（如 “验证成功？”））
* 初始/终止节点 （实心圆 / 实心圆 + 边框）
# 状态图
## 组成
* 状态（圆角矩形（如 “待付款”“已发货”））
* 初始状态（实心圆）
* 终止状态（实心圆 + 边框）
* 转化（带箭头的线，标注触发事件（如 “支付完成”））
