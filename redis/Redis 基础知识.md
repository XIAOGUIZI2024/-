# 常见的数据类型
* String：这个是最常见的数据类型，能够存储数字，字符串，二进制数，最大的存储内存是512MB，主要是用做缓存，计数器，分布式锁
* List：一个有序的字符串列表，底层是双向链表，支持两端的操作，主要是用作在消息队列，时间轴，最新的信息列表
* Set：无序不可以重复的集合，主要是用作去重
* Hash：一个建值对集合存储结构化数据（对象的信息）
* Zset：带有一个score分数进行排序的set，主要是用作排行榜
* BitMap：用位来存储数据的，每一个bit只占0或者1，空间的使用程度很高，主要是用在用户的签到，在线状态统计
* Stream：5.0版本引入的，专门为消息队列设计的，两个关键的特点，其一会自动生成全局唯一的信息ID，其二可以信息持久化，主要是用于消息队列
* GEO：用来存储地理的位置信息，支持经纬对和空间存储与查询，主要是附件的人/店铺，地理位置的标记
	* HyperLogLog：概率算法，用极小内存（固定12KB）统计巨大数据集的基数（去重计数），是一个概率性数据结构，专门用来估算基数，主要是用于搜索关键词去重计算~！~
