## 一、接口概述

### 接口功能

调用火山引擎方舟 `doubao-seedance-2-0-260128` 模型，**根据文本指令 + 参考图片 / 视频 / 音频**生成定制化视频，支持第一视角构图、背景音乐绑定、分镜脚本、音频生成等定制化参数配置。

### 基础信息

- **请求域名**：`https://ark.cn-beijing.volces.com/api/v3`
- **认证方式**：API Key 鉴权（环境变量 `ARK_API_KEY`）
- **SDK 依赖**：`volcengine-python-sdk[ark]`
- **任务模式**：异步任务（提交后轮询查询结果）

## 二、创建视频生成任务接口

### 1. 接口地址

`POST /content_generation/tasks/create`

### 2. 请求头（SDK 自动封装，无需手动传参）

表格

| 参数名           | 类型     | 必填  | 说明                |
| ------------- | ------ | --- | ----------------- |
| Authorization | string | 是   | Bearer ${API_KEY} |
| Content-Type  | string | 是   | application/json  |
|               |        |     |                   |

### 3. 请求参数（Body）

表格

|参数名|类型|必填|说明|示例值|
|---|---|---|---|---|
|model|string|是|模型 ID，固定使用当前版本|`doubao-seedance-2-0-260128`|
|content|array[object]|是|生成内容主体，包含文本 + 多模态参考资源|见下方子参数|
|generate_audio|boolean|否|是否自动生成音频（背景音 / 人声）|`true`|
|ratio|string|否|视频比例|`16:9` / `9:16` / `1:1`|
|duration|int|否|视频时长，单位：秒|8 / 11|
|watermark|boolean|否|是否添加平台水印|`true` / `false`|

#### content 数组子参数（多模态资源）

content 为**有序数组**，支持 `text`/`image_url`/`video_url`/`audio_url` 四种类型，**必须按业务需求组合传入**：

##### （1）文本指令（必填，第一个元素）

表格

|参数名|类型|必填|说明|
|---|---|---|---|
|type|string|是|固定值：`text`|
|text|string|是|视频生成详细指令（分镜、构图、音效、画面要求）|

##### （2）参考图片（可选，可传多张）

表格

|参数名|类型|必填|说明|
|---|---|---|---|
|type|string|是|固定值：`image_url`|
|image_url.url|string|是|参考图片公网 URL|
|role|string|是|固定值：`reference_image`|

##### （3）参考视频（可选）

表格

|参数名|类型|必填|说明|
|---|---|---|---|
|type|string|是|固定值：`video_url`|
|video_url.url|string|是|参考视频公网 URL|
|role|string|是|固定值：`reference_video`|

##### （4）参考音频（可选）

表格

|参数名|类型|必填|说明|
|---|---|---|---|
|type|string|是|固定值：`audio_url`|
|audio_url.url|string|是|背景音乐 / 音频公网 URL|
|role|string|是|固定值：`reference_audio`|

## 三、查询任务状态接口

### 1. 接口地址

`GET /content_generation/tasks/get`

### 2. 请求参数（Query/SDK 参数）

表格

|参数名|类型|必填|说明|
|---|---|---|---|
|task_id|string|是|创建任务返回的任务 ID|

### 3. 响应状态说明

表格

|状态值|含义|处理逻辑|
|---|---|---|
|pending|任务排队中|继续轮询|
|running|任务执行中|继续轮询|
|succeeded|任务成功|获取视频结果|
|failed|任务失败|查看错误信息|

## 五、字段说明总结

1. **核心必传**：`model` + `content`（必须包含文本指令）
2. **多模态增强**：参考图片 / 视频 / 音频可大幅提升生成效果
3. **视频控制**：`ratio`（比例）、`duration`（时长）、`watermark`（水印）
4. **音频控制**：`generate_audio` 开启后自动合成背景人声
