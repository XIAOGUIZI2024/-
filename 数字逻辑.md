* 十进制 <-> 二进制
```java
十进制转化为二进制
整数部分 - 除2取余法 从下向上对
(13)D
13/2 -> 1
6/2 -> 0
3/2 -> 1
1/2 -> 1
(1101)B

小数部分 - 乘2取整法 从上向下
(0.625)D
0.625*2 -> 1
0.25*2 -> 0
0.5*2 -> 1
(0.101)B

二进制转化为十进制
小数点左边：从右向左，2⁰, 2¹, 2², ...
    
小数点右边：从左向右，2⁻¹, 2⁻², 2⁻³, ...
(1101.101)₂
1 + 0 + 4 + 8 = 13
0.5 + 0.125 = 0.625
(13.625)D
```

# 最小项
**都为与项 而且每一项都要只可以包含一个元素**
* 任意一个最小项，只有一组变量取值使得它的值为1
* 不同的最小项，使它的值为1的那一组变量取值不同
* 任意两个最小项的乘积为0
* 对于变量的任意一组的取值，全体最小项之和为1
**最小项的编号** 原变量为 **1** 非变量为 **0**
