# 进制数的转化
* 八进制  <->  二进制
```java
三位一并法 421

二进制转化为八进制
(011 010 111.0111)B 
011 -> 3
010 -> 2
111 -> 7
.011 -> 3
.100 -> 4
(327.34)O

八进制转化为二进制
(45.6)B
4 -> 100
5 -> 101
.6 ->110
(100 101.11)B 
```
* 十进制 <-> 二进制
```java
十进制转化为二进制
整数部分 - 除2取余法 从下向上对
(13)D
13/2 -> 1
6/2 -> 0
3/2 -> 1
1/2 -> 1
(1101)B

小数部分 - 乘2取整法 从上向下
(0.625)D
0.625*2 -> 1
0.25*2 -> 0
0.5*2 -> 1
(0.101)B

二进制转化为十进制
小数点左边：从右向左，2⁰, 2¹, 2², ...
    
小数点右边：从左向右，2⁻¹, 2⁻², 2⁻³, ...
(1101.101)₂
1 + 0 + 4 + 8 = 13
0.5 + 0.125 = 0.625
(13.625)D
```
